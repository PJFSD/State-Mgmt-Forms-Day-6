# Client Portal (React + FastAPI + MySQL)


# Project Overview :-

The Client Portal is a full-stack web application built using React for the frontend, FastAPI for the backend, and MySQL for the database.

It allows users to create and view projects through a simple web interface. The React frontend sends requests to the FastAPI backend using Axios, and the backend stores the project information in the MySQL database.

This project shows how frontend, backend, and database work together in a full-stack application.


# Built With :

 # Frontend -
1. React.js
2. Axios
3. CSS

# Backend -
1. FastAPI
2. Python

# Database -
1. MySQL


# Features :
Create new Projects.
View projects in a table.
React form handling using useState.
Fetch data from FastAPI API.
Store data in MySQL database.
Error handling using React Error Boundary.


# Backend Setup :

# How to Run -
1. cd backend
2. python -m uvicorn main:app --reload
3. Open API documentation - http://127.0.0.1:8000/docs


# Frontend Setup :

# How to Run -
1. cd frontend
2. npm start
3. Open application - http://localhost:3000
